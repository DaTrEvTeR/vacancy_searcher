# **Vacancy Searcher**

---

---

## **Dev**

Make sure you have [Poetry](https://python-poetry.org/) installed. To install all project dependencies, run:

```shell
poetry install --no-root
```

Activate environment shell by:

```shell
poetry shell
```

---

## Run

Use next command to run script:

```shell
python main.py
```
